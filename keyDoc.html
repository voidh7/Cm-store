Documentação - Sistema de Chaves de Acesso

Nota: Essa API foi pensada especialmente para desenvolvedores que usam a engine Pocket Code, facilitando a integração de verificação de chaves em seus projetos.

Essa ferramenta foi feita pra você que quer liberar qualquer conteúdo online só pra quem tiver uma chave de acesso válida.

1. Criando uma chave

Acesse o painel em:

https://cm-store.vercel.app/keys.html

Clique em "Gerar nova chave". Você receberá:

  • ID (ex: abc123)
  • Chave (ex: XPTO999)

2. Verificando a chave

Use esse endpoint para verificar se a chave está válida e ainda não foi usada:

https://caiolibs.vercel.app/api/keyChecked.js?id=SEU_ID&chave=SUA_CHAVE

Exemplo:

https://caiolibs.vercel.app/api/keyChecked.js?id=abc123&chave=XPTO999

3. Código JavaScript de exemplo

fetch("https://caiolibs.vercel.app/api/keyChecked.js?id=abc123&chave=XPTO999")
  .then(res => res.text())
  .then(resposta => {
    if (resposta === "true") {
      alert("Acesso liberado!");
    } else {
      alert("Chave inválida ou já usada!");
    }
  });

Regras

  • Chave válida = true
  • Chave inválida ou já usada = false
  • Cada chave só pode ser usada uma vez

Possíveis usos

  • Downloads exclusivos
  • Links secretos
  • Conteúdo premium
  • Ativar recursos em jogos ou apps
