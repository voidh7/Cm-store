Termos de Uso

Natureza do Projeto: A CM Store é um projeto desenvolvido exclusivamente para fins de estudo e prática de programação web. Esta plataforma não possui fins comerciais e foi criada como exercício educacional.
Disponibilidade: Este site pode ser retirado do ar a qualquer momento, sem aviso prévio, por motivos técnicos, de estudo ou qualquer outro motivo. Todos os dados e jogos armazenados podem ser permanentemente apagados em caso de problemas com o banco de dados ou como parte do ciclo de desenvolvimento.

Para Desenvolvedores

A CM Store é uma plataforma experimental para publicação de jogos, criada como projeto de estudo. Não nos responsabilizamos pelo conteúdo publicado.

1. Natureza Temporária: Os jogos publicados podem ser removidos a qualquer momento conforme as necessidades do projeto de estudo.

2. Backup Responsável: Recomendamos que desenvolvedores mantenham backups próprios de seus jogos, pois não garantimos a permanência dos arquivos na plataforma.

3. Conteúdo Proibido: Não são permitidos jogos que contenham:

  • Material ilegal, pirata ou violador de direitos autorais
  • Conteúdo ofensivo, discriminatório ou de ódio
  • Vírus, malware ou qualquer código malicioso

4. Selo de confiança: Recomendamos que desenvolvedores mantenham contato com a equipe cm store para continuar Obtendo o selo de confiança.
Clique aqui e fale com o dev da cmstore sobre o assunto.

Para Jogadores

A CM Store é um projeto experimental de estudo. Todo download é feito por sua conta e risco.

1. Disponibilidade Limitada: Os jogos podem se tornar indisponíveis a qualquer momento sem aviso prévio.

2. Segurança: Como projeto de estudo, recomendamos extremo cuidado:

  • Verifique se o jogo tem "selo de verificado" ao lado do nome (Geralmente esses jogos são revisados)
  • Verifique os comentários antes de baixar
  • Use antivírus atualizado
  • Não execute arquivos com permissões administrativas

Disposições Gerais

1. Natureza Educacional: Esta plataforma não oferece garantias de qualquer tipo, sendo um projeto acadêmico/educacional.

2. Dados Temporários: Todos os dados armazenados são considerados temporários e podem ser apagados durante o processo de desenvolvimento.

3. Isenção Total: Por se tratar de projeto de estudo, os criadores estão isentos de qualquer responsabilidade sobre o uso da plataforma.
