Save GC - Documentação

Save GC é uma API hospedada na CM Store (uma loja de jogos feita por Caio Multiversando). Ela pertence à GC Editora, criadora do Pocket Race, e foi criada para substituir o bloco de salvar em arquivo no Pocket Code.

Como usar?

Para salvar dados, informe:

  • O nome do jogo
  • O nome do arquivo
  • Um ID para o jogador

Por que cada informação é necessária?

Nome do Jogo

Para a API saber para onde enviar os dados no armazém.

Nome do Arquivo

Porque há muitos arquivos diferentes. Precisamos saber qual deve ser usado.

ID do Jogador

Para diferenciar jogadores e seus dados.

Dica: use um número aleatório entre 1 e 10.000 como ID.

Salvando arquivo

cm-store.vercel.app/api/arquivo.js?acao=salvar&nomeJogo=FreeFire&nomeArquivo=pontos&id=caio&valor=743

Isso envia o valor 743 para o jogo FreeFire, no arquivo pontos do jogador Caio.

Usando no Pocket Code

Use o bloco "enviar solicitação para web" com a URL acima.

Substitua os valores pelos do seu jogo.

Use & para separar os parâmetros na URL.

Lendo dados

Altere acao=salvar para acao=ler e remova valor=.

cm-store.vercel.app/api/arquivo.js?acao=ler&nomeJogo=FreeFire&nomeArquivo=pontos&id=caio

Juntando variáveis na URL

Use a função JUNTAR() no Pocket Code para combinar texto e variáveis.

Juntar('Olá', nome)
